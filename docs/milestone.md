

### **[콘서트 예약 시스템] 시나리오 설계**
- 총 3주로 진행예정



**1. 시나리오 선정**
- 프로젝트 Milestone 설정
- 요구사항 분석 및 자료 작성 (시퀀스 다이어그램, 플로우 차트 등 )
- ERD 설계 자료 작성
- API 명세 및 Mock API 작성
- 기본 패키지 구조 설계
  
**2. 예약 시스템 시나리오 구현**
- 유저 토큰 발급 API
- 예약 가능 날짜 / 좌석 API
- 좌석 예약 요청 API
- 잔액 충전 / 조회 API
- 결제 API
  
**3. 예약 시스템 시나리오 고도화**
- 유저 토큰 발급 API 고도화
- 예약 가능 날짜 / 좌석 API 고도화
- 좌석 예약 요청 API 고도화
- 잔액 충전 / 조회 API 고도화
